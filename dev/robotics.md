## Glossary

- Closed-loop system
    - Output is fed back into the input (uses feedback)
- Open-loop system
    - Output is not fed back (no feedback)
- Mapping
    - To build a map of the environment
- Localization
    - To estimate the object's pose within a map
- pose
    - x, y, z
    - orientation
